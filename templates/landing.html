Password Hashing System

A safe authentication system constructed using contemporary web technologies. Our platform provides a fully functional sign-up and login mechanism that uses the SHA-256 technique for password hashing to protect user data.

The technology transforms each password into a distinct fixed-length hash rather than storing them in plain text, rendering them unintelligible even in the event that the database is stolen.

We integrated our authentication logic, handled forms, and routed traffic using Flask, a lightweight Python web framework, on the backend. Clean HTML and CSS are used to construct the frontend, and SQLite is used to safely store the data.

The three specialist divisions that make up our team—Requirements, Documentation, and Coding—all contribute to the project's conception, creation, and implementation. We integrated user experience, teamwork, and security into one seamless web application.
